# UAV Vehicle Detection using YOLOv11-OBB

A web-based tool for detecting vehicles from UAV imagery using **YOLOv11 with Oriented Bounding Boxes (OBB)**. The system automatically fetches tiles from the Taiwan UAV WMTS service and performs vehicle detection in real-time. Detection results are visualized on a Leaflet map and can be downloaded as GeoJSON.

---

## Features

- Detect **small and large vehicles** from UAV imagery.
- Handles **oriented bounding boxes (OBB)** for accurate detection.
- Interactive **web interface** using Leaflet for drawing detection areas.
- Real-time vehicle detection over user-defined regions.
- Export detection results as **GeoJSON**.
- Integrates **YOLOv11 OBB model** with **Python FastAPI backend**.
- Automatically fetches tiles from Taiwan's **UAV WMTS**.

---

## Demo

Once the server is running, open the public URL generated by **ngrok**.  

1. Draw a rectangle on the map to define the detection area.  
2. Click **Detect Vehicles** to run detection.  
3. Click **Download GeoJSON** to save results.  
4. Use **Clear Detections** to reset.
